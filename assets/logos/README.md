## 💡 Guía de Selección

**¿Cuál archivo debo usar?**

✅ **Usa PNG cuando:**
- Necesites transparencia
- Sea para web o redes sociales
- Requieras máxima calidad
- El fondo varíe

✅ **Usa JPG cuando:**
- Necesites fondos sólidos
- Sea para impresión
- Quieras menor peso de archivo
- Compatibilidad universal

✅ **Usa Grayscale cuando:**
- Material corporativo B&W
- Documentos oficiales
- Impresión en blanco y negro

✅ **Usa Inverted cuando:**
- Fondos oscuros o de colores
- Efectos visuales especiales
